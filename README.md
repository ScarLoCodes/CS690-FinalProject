# Wellness Tracker
<img width="570" height="299" alt="image" src="https://github.com/user-attachments/assets/3223fa65-86ac-4fc6-88c2-3576ffcbc306" />

The wellness tracker is a C# Console app that allows you to track Fitness, Nutrition, and Hydration goals.
You can enter a goal you'd like to set, an end date for the goal, and begin logging activities that
 contribute to it. You can also set reminders to help you remember to log certain activities.
